### Typesetting

The content of the notes is formatted in accordance with [Chinese Copywriting Guidelines](https://github.com/sparanoid/chinese-copywriting-guidelines/blob/master/README.zh-CN.md) to ensure the readability of the content.

Instead of using `![]()` to reference images, use the `<img>` tag. On the one hand, it is to be able to control the image to be displayed at an appropriate size, and on the other hand, [GFM](https://github.github.com/gfm/) does not support the `<center> ![]() </center>` method to center the image. You can only use `<div align="center"> <img src=""/> </div>` to achieve the centering effect.

Online typesetting tool: [Text-Typesetting](https://github.com/CyC2018/Text-Typesetting).
