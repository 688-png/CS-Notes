## Title description

We can use a small rectangle of 2\*1 to cover the larger rectangle horizontally or vertically. How many ways are there to cover a large rectangle of 2\*n with n small rectangles of 2\*1 without overlapping?

To cover a large rectangle of 2\*n, you can first cover the rectangle of 2\*1 and then the rectangle of 2\*(n-1); or first cover the rectangle of 2\*2 and then cover the rectangle of 2\*(n-2). And the rectangle covering 2\*(n-1) and 2\*(n-2) can be regarded as a sub-problem. The recursive formula for this problem is as follows:

```java
public int rectCover(int n) {
    if (n <= 2)
        return n;
    int pre2 = 1, pre1 = 2;
    int result = 0;
    for (int i = 3; i <= n; i++) {
        result = pre2 + pre1;
        pre2 = pre1;
        pre1 = result;
    }
    return result;
}
```
