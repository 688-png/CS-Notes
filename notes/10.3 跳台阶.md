When n = 2, there are two jumping methods:

To jump n steps, you can jump 1 step first, and then jump n-1 steps; or jump 2 steps first, and then jump n-2 steps. The jumping method of n-1 and n-2 steps can be regarded as a sub-problem. The recursive formula of this problem is:

<div align="center"> <img src="https://cs-notes-1256109796.cos.ap-guangzhou.myqcloud.com/508c6e52-9f93-44ed-b6b9-e69050e14807.jpg" width="350px"> </div><br>

```java
public int JumpFloor(int n) {
    if (n <= 2)
        return n;
    int pre2 = 1, pre1 = 2;
    int result = 0;
    for (int i = 2; i < n; i++) {
        result = pre2 + pre1;
        pre2 = pre1;
        pre1 = result;
    }
    return result;
}
```
