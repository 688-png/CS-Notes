# 14. Cut the rope

## Question link

## Title description

Cut a rope into multiple segments so that the product of the lengths of each segment is the maximum.

```html
n=2
return 1 (2 = 1 + 1)

n=10
return 36 (10 = 3 + 3 + 4)
```

## Problem-solving ideas

### Greedy

Cut as many length 3 ropes as possible, and no length 1 ropes are allowed. If it does, take a piece of rope that has been cut to length 3, reassemble it with the rope of length 1, and cut them into two pieces of length 2 rope. The following is the proof process.

Split the rope into 1 and n-1, then 1(n-1)-n=-1\<0, that is, the product after splitting must be smaller, so there cannot be a rope with a length of 1.

Split the rope into 2 and n-2, then 2(n-2)-n = n-4. When n\>=4, the product obtained by splitting in this way will be larger than without splitting.

Split the rope into 3 and n-3, then 3(n-3)-n = 2n-9, and the effect is better when n\>=5.

Splitting the rope into 4 and n-4 has the same effect as splitting it into 2 since 4=2\*2.

Split the rope into 5 and n-5, because 5=2+3 and 5\<2\*3, so the rope of 5 cannot appear, but split it into 2 and 3 as much as possible.

Split the rope into 6 and n-6, because 6=3+3 and 6\<3\*3, so the rope of 6 cannot appear, but split into 3 and 3. Here 6 can also be split into 6=2+2+2, but 3(n - 3) - 2(n - 2) = n - 5 \>= 0. In the case of n\>=5, splitting the rope into 3 is better than splitting into 2.

If you continue to split the rope into larger ropes, you will find that the effect is worse than splitting it into 2 and 3. Therefore, we only consider splitting the rope into 2 and 3, and give priority to splitting it into 3. When the rope length n is equal to 4, that is, 3+1 appears. At this time, it can only be split into 2+2.

```java
public int cutRope(int n) {
    if (n < 2)
        return 0;
    if(n==2)
        return 1;
    if(n==3)
        return 2;
    int timesOf3 = n / 3;
    if (n - timesOf3 * 3 == 1)
        timesOf3--;
    int timesOf2 = (n - timesOf3 * 3) / 2;
    return (int) (Math.pow(3, timesOf3)) * (int) (Math.pow(2, timesOf2));
}
```

### Dynamic programming

```java
public int cutRope(int n) {
    int[] dp = new int[n + 1];
    dp[1] = 1;
    for (int i = 2; i <= n; i++)
        for (int j = 1; j < i; j++)
            dp[i] = Math.max(dp[i], Math.max(j * (i - j), dp[j] * (i - j)));
    return dp[n];
}
```
