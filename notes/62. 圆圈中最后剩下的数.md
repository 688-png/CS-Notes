# 62. The last remaining number in the circle

## Title description

Have the children form a large circle. Then, randomly assign a number m and let the child numbered 0 start counting. Each time the child who calls m-1 will go out of the queue to sing a song, and then he can choose any gift in the gift box, and will not return to the circle. Starting from the next child, he will continue to count 0...m-1... and this goes on... until the last child is left, and there is no need to perform.

## Problem-solving ideas

Joseph ring, the solution of a circle of length n can be seen as a solution of length n-1 plus the length of the number m. Because it is a circle, we need to take the remainder of n in the end.

```java
public int LastRemaining_Solution(int n, int m) {
    if (n == 0) /* Special input processing */
        return -1;
    if (n == 1) /* recursive return condition */
        return 0;
    return (LastRemaining_Solution(n - 1, m) + m) % n;
}
```
