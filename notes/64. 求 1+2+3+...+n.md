## Problem-solving ideas

The most important thing when using a recursive solution is to specify the return condition, but this question cannot directly use the if statement to specify the return condition.

Conditions and && have a short-circuit principle, that is, the second conditional statement will not be executed if the first conditional statement is false. Using this feature, the recursive return condition is negated and used as the first conditional statement of &&, and the recursive body is converted into the second conditional statement. Then when the recursive return condition is true, the main part of the recursive will not be executed and the recursive return will occur.

The recursive return condition of this question is n \<= 0, and after negation, it is n \> 0; the main part of the recursion is sum += Sum_Solution(n - 1), and when converted into a conditional statement, it is (sum += Sum_Solution(n - 1)) \> 0.

```java
public int Sum_Solution(int n) {
    int sum = n;
    boolean b = (n > 0) && ((sum += Sum_Solution(n - 1)) > 0);
    return sum;
}
```
