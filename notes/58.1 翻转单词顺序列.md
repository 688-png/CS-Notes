## Problem-solving ideas

Flip each word first, then flip the entire string.

The question should have an implicit condition that no extra space should be used. Although the input parameter of Java's question is of String type, and a character array needs to be created first to make the space complexity O(N), the correct parameter type should be the same as the original book, which is a character array, and only the space of the character array can be used. Any solution that uses extra space will be greatly discounted during the interview, including recursive solutions.

```java
public String ReverseSentence(String str) {
    int n = str.length();
    char[] chars = str.toCharArray();
    int i = 0, j = 0;
    while (j <= n) {
        if (j == n || chars[j] == ' ') {
            reverse(chars, i, j - 1);
            i = j + 1;
        }
        j++;
    }
    reverse(chars, 0, n - 1);
    return new String(chars);
}

private void reverse(char[] c, int i, int j) {
    while (i < j)
        swap(c, i++, j--);
}

private void swap(char[] c, int i, int j) {
    char t = c[i];
    c[i] = c[j];
    c[j] = t;
}
```
