## Title description

Use two stacks to implement a queue and complete the push and pop operations of the queue.

## Problem-solving ideas

The in stack is used to handle push operations, and the out stack is used to handle pop operations. After an element is entered into the in stack, the order in which it is popped out of the stack is reversed. When an element is to be popped from the stack, it needs to be entered into the out stack first. At this time, the order in which the elements are popped out of the stack is reversed again. Therefore, the order in which the elements are popped out of the stack is the same as the initial order in which they were pushed into the stack. The element that enters first exits first. This is the order of the queue.

```java
Stack<Integer> in = new Stack<Integer>();
Stack<Integer> out = new Stack<Integer>();

public void push(int node) {
    in.push(node);
}

public int pop() throws Exception {
    if (out.isEmpty())
        while (!in.isEmpty())
            out.push(in.pop());

    if (out.isEmpty())
        throw new Exception("queue is empty");

    return out.pop();
}
```
