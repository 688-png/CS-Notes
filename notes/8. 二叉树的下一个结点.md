## Title description

Given a binary tree and a node in it, find the next node in the inorder traversal sequence and return . Note that the nodes in the tree not only contain left and right child nodes, but also pointers to parent nodes.

```java
public class TreeLinkNode {

    int val;
    TreeLinkNode left = null;
    TreeLinkNode right = null;
    TreeLinkNode next = null; // Pointer to parent node

    TreeLinkNode(int val) {
        this.val = val;
    }
}
```

## Problem-solving ideas

Let's first review the process of in-order traversal: first traverse the left subtree of the tree, then traverse the root node, and finally traverse the right subtree. So the leftmost node is the first node in inorder traversal.

```java
void traverse(TreeNode root) {
    if (root == null) return;
    traverse(root.left);
    visit(root);
    traverse(root.right);
}
```

```java
public TreeLinkNode GetNext(TreeLinkNode pNode) {
    if (pNode.right != null) {
        TreeLinkNode node = pNode.right;
        while (node.left != null)
            node = node.left;
        return node;
    } else {
        while (pNode.next != null) {
            TreeLinkNode parent = pNode.next;
            if (parent.left == pNode)
                return parent;
            pNode = pNode.next;
        }
    }
    return null;
}
```
