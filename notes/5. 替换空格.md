# 5. Replace spaces

## Title description


Replaces spaces in a string with "%20".

```text
Input:
"A B"

Output:
"A%20B"
```

## Problem-solving ideas

① Pad any characters at the end of the string so that the length of the string is equal to the length after replacement. Because a space is replaced by three characters (%20), when a space is traversed, two arbitrary characters need to be padded at the end.

② Let P1 point to the original end position of the string, and P2 point to the current end position of the string. P1 and P2 are traversed from back to front. When P1 traverses to a space, the position pointed to by P2 needs to be filled with 02% in turn (note that it is in reverse order), otherwise it will be filled with the value of the character pointed to by P1. The purpose of traversing from back to front is to change the content pointed by P2 without affecting the content of the original string traversed by P1.

③ When P2 encounters P1 (P2 \<= P1), or the traversal ends (P1 \< 0), exit.

```java
public String replaceSpace(StringBuffer str) {
    int P1 = str.length() - 1;
    for (int i = 0; i <= P1; i++)
        if (str.charAt(i) == ' ')
            str.append(" ");

    int P2 = str.length() - 1;
    while (P1 >= 0 && P2 > P1) {
        char c = str.charAt(P1--);
        if (c == ' ') {
            str.setCharAt(P2--, '0');
            str.setCharAt(P2--, '2');
            str.setCharAt(P2--, '%');
        } else {
            str.setCharAt(P2--, c);
        }
    }
    return str.toString();
}
```
