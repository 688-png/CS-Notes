# 67. Convert string to integer

## Title description

Convert a string into an integer. If the string is not a legal value, 0 is returned. It is required that the library function for converting a string into an integer cannot be used.

```html
Iuput:
+2147483647
1a33

Output:
2147483647
0
```

## Problem-solving ideas

```java
public int StrToInt(String str) {
    if (str == null || str.length() == 0)
        return 0;
    boolean isNegative = str.charAt(0) == '-';
    int ret = 0;
    for (int i = 0; i < str.length(); i++) {
        char c = str.charAt(i);
        if (i == 0 && (c == '+' || c == '-')) /* Sign determination */
            continue;
        if (c < '0' || c > '9') /* Illegal input */
            return 0;
        ret = ret * 10 + (c - '0');
    }
    return isNegative ? -ret : ret;
}
```
