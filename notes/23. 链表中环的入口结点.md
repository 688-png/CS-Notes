# 23. Entry node of the ring in the linked list

[NowCoder](https://www.nowcoder.com/practice/253d2c59ec3e4bc68da16833f79a38e4?tpId=13&tqId=11208&tPage=1&rp=1&ru=/ta/coding-interviews&qru=/ta/coding-interviews/question-ranking&from=cyc_github)

## Title description

A linked list contains a ring. Please find the entry node of the ring in the linked list. No additional space is required.

## Problem-solving ideas

Use dual pointers, a fast pointer that moves two nodes at a time, and a slow pointer that moves one node at a time. Because there is a ring, the two pointers must meet at a node in the ring.

Assume that the ring entry node is y1 and the node where they meet is z1.

Assuming that the fast pointer fast goes around N times in the circle, the total path length is x+Ny+(N-1)z. The reason z is (N-1) times is because the fast and slow pointers have finally met at the z1 node, and there is no need to move further.

The total path length of the slow pointer is x+y.

Since fast pointers are twice as long as slow pointers, x+Ny+(N-1)z = 2(x+y).

What we are looking for is the ring entrance node y1, which can also be regarded as finding the value of length x, so we first decompose the above equivalent value into related to x: x=(N-2)y+(N-1)z.

There is no strong rule for the above equivalent values, but we can find that y+z is the total length of the ring, so we decompose the above equation again: x=(N-2)(y+z)+z. The left side of this equation is the length from the starting point x1 to the ring entrance node y1, while the right side is the length of walking through (N-2) circles in the ring, and then walking through the length z from the meeting point z1. At this point we can find that if the two pointers start from the starting point x1 and the meeting point z1 at the same time, and only travel one distance each time, then they will eventually meet at the ring entrance node.

```java
public ListNode EntryNodeOfLoop(ListNode pHead) {
    if (pHead == null || pHead.next == null)
        return null;
    ListNode slow = pHead, fast = pHead;
    do {
        fast = fast.next.next;
        slow = slow.next;
    } while (slow != fast);
    fast = pHead;
    while (slow != fast) {
        slow = slow.next;
        fast = fast.next;
    }
    return slow;
}
```
