# 22. The Kth node from the last in the linked list

## Problem-solving ideas

Let the length of the linked list be N. Set two pointers P1 and P2, first let P1 move K nodes, then there are N - K nodes that can be moved. At this time, let P1 and P2 move at the same time. You can know that when P1 moves to the end of the linked list, P2 moves to the N-Kth node, which is the Kth node from the bottom.

```java
public ListNode FindKthToTail(ListNode head, int k) {
    if (head == null)
        return null;
    ListNode P1 = head;
    while (P1 != null && k-- > 0)
        P1 = P1.next;
    if(k>0)
        return null;
    ListNode P2 = head;
    while (P1 != null) {
        P1 = P1.next;
        P2 = P2.next;
    }
    return P2;
}
```
