## Title description

All numbers in an array of length n are in the range 0 to n-1. Some numbers in the array are repeated, but we don't know how many numbers are repeated, nor do we know how many times each number is repeated. Please find any repeated number in the array.

```html
Input:
{2, 3, 1, 0, 2, 5}

Output:
2
```

## Problem-solving ideas

The required time complexity is O(N) and the space complexity is O(1). Therefore, the sorting method cannot be used, nor can additional tag arrays be used.

For this problem where the array elements are in the range [0, n-1], the element with value i can be adjusted to the i-th position to solve. During the adjustment process, if there is already an element with value i at the i-th position, you can know that the i value is repeated.

Take (2, 3, 1, 0, 2, 5) as an example. When traversing to position 4, the number at that position is 2, but there is already a value of 2 at the second position, so you can know that 2 is repeated:

```java
public int duplicate(int[] nums) {
    for (int i = 0; i < nums.length; i++) {
        while (nums[i] != i) {
            if (nums[i] == nums[nums[i]]) {
                return nums[i];
            }
            swap(nums, i, nums[i]);
        }
        swap(nums, i, nums[i]);
    }
    return -1;
}

private void swap(int[] nums, int i, int j) {
    int t = nums[i];
    nums[i] = nums[j];
    nums[j] = t;
}

```
