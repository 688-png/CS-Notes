# 56. Numbers that appear only once in the array

## Question link

[Niuke.com](https://www.nowcoder.com/practice/389fc1c3d3be4479a154f63f495abff8?tpId=13&tqId=11193&tab=answerKey&from=cyc_github)

## Title description

Except for two numbers in an integer array, all other numbers appear twice. Find these two numbers.

## Problem-solving ideas

The result of XORing two equal elements is 0, and the result of XORing 0 with any number x is x.

Perform the XOR operation on all elements of the array given in this question, and the result obtained is the XOR result of two non-duplicate elements. For example, for the array [x,x,y,y,z,k], x^x^y^y^z^k = 0^y^y^z^k = y^y^z^k = 0^z^k = z^k.

Two unequal elements must be different in bit-level representation, so the result of XORing these two elements, diff, must not be 0. The bitwise operation diff & -diff can obtain the rightmost bit of 1 in the bit-level representation of diff, let diff = diff & -diff. Using diff as the basis for distinguishing two elements, there must be one element that XORs diff with a result of 0, and the other result is non-0. Assume that the two unequal elements are z and k. Traverse all elements of the array and determine whether the XOR result of the element and diff is 0. If so, XOR the element with z and assign it to z. Otherwise, XOR it with k and assign it to k. Equal elements in the array must be XORed with z or k at the same time, instead of one being XORed with z and the other being XORed with k. And the result of the XOR of these equal elements is 0, so in the end z and k are just the results of the XOR of two unequal elements with 0, which is the two unequal elements themselves.

In the following solution, the first element of the num1 and num2 arrays is used to hold the return value... This method of returning values ​​is not recommended in actual development.

```java
public int[] FindNumsAppearOnce (int[] nums) {
    int[] res = new int[2];
    int diff = 0;
    for (int num : nums)
        diff ^= num;
    diff &= -diff;
    for (int num : nums) {
        if ((num & diff) == 0)
            res[0] ^= num;
        else
            res[1] ^= num;
    }
    if (res[0] > res[1]) {
        swap(res);
    }
    return res;
}

private void swap(int[] nums) {
    int t = nums[0];
    nums[0] = nums[1];
    nums[1] = t;
}
```
