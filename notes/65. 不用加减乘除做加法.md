## Title description

Write a function to find the sum of two integers. It is required that the four arithmetic symbols +, -, \*, and / are not allowed.

## Problem-solving ideas

a ^ b represents the sum of two numbers without considering carry. (a & b) \<\< 1 is carry.

The reason why the recursion will terminate is that (a & b) \<\< 1 will have one more 0 on the rightmost side. If the recursion continues, the rightmost 0 in the carry will gradually increase, and finally the carry will become 0, and the recursion terminates.

```java
public int Add(int a, int b) {
    return b == 0 ? a : Add(a ^ b, (a & b) << 1);
}
```
