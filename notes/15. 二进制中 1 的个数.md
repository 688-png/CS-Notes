## Title description

Input an integer and output the number of 1's in the binary representation of the number.

### Problem-solving ideas

The n&(n-1) bit operation sets the lowest bit, 1, in n's bit-level representation to 0. Keep setting 1 to 0 until n is 0. Time complexity: O(M), where M represents the number of 1's.

```java
public int NumberOf1(int n) {
    int cnt = 0;
    while (n != 0) {
        cnt++;
        n &= (n - 1);
    }
    return cnt;
}
```
