## Title description

A frog can jump up 1 step at a time, it can jump up 2 steps... it can also jump up n steps. Find the total number of ways the frog can jump up an n-level staircase.

## Problem-solving ideas

### Dynamic programming

```java
public int jumpFloorII(int target) {
    int[] dp = new int[target];
    Arrays.fill(dp, 1);
    for (int i = 1; i < target; i++)
        for (int j = 0; j < i; j++)
            dp[i] += dp[j];
    return dp[target - 1];
}
```

### Mathematical derivation

Jumping up the n-1 steps, you can jump 1 step up from n-2 level, or you can jump 2 steps up from n-3 level..., then

```
f(n-1) = f(n-2) + f(n-3) + ... + f(0)
```

Similarly, to jump up n steps, you can jump 1 step up from n-1 level, or you can jump 2 steps up from n-2 level... , then

```
f(n) = f(n-1) + f(n-2) + ... + f(0)
```

To sum up, we can get

```
f(n) - f(n-1) = f(n-1)
```

That is

```
f(n) = 2*f(n-1)
```

So f(n) is a geometric sequence

```source-java
public int JumpFloorII(int target) {
    return (int) Math.pow(2, target - 1);
}
```
