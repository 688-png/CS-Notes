## Title description

The binary tree is reconstructed based on the results of pre-order traversal and in-order traversal of the binary tree. It is assumed that the results of the input pre-order traversal and in-order traversal do not contain repeated numbers.

## Problem-solving ideas

The first value of the pre-order traversal is the value of the root node. This value is used to divide the in-order traversal result into two parts. The left part is the in-order traversal result of the left subtree of the tree, and the right part is the result of the in-order traversal of the right subtree of the tree. Then recursively solve the left and right subtrees respectively.

```java
//Cache the index corresponding to each value of the in-order traversal array
private Map<Integer, Integer> indexForInOrders = new HashMap<>();

public TreeNode reConstructBinaryTree(int[] pre, int[] in) {
    for (int i = 0; i < in.length; i++)
        indexForInOrders.put(in[i], i);
    return reConstructBinaryTree(pre, 0, pre.length - 1, 0);
}

private TreeNode reConstructBinaryTree(int[] pre, int preL, int preR, int inL) {
    if (preL > preR)
        return null;
    TreeNode root = new TreeNode(pre[preL]);
    int inIndex = indexForInOrders.get(root.val);
    int leftTreeSize = inIndex - inL;
    root.left = reConstructBinaryTree(pre, preL + 1, preL + leftTreeSize, inL);
    root.right = reConstructBinaryTree(pre, preL + leftTreeSize + 1, preR, inL + leftTreeSize + 1);
    return root;
}
```
