## Title description

Find two numbers in a sorted array such that their sum is the given number S. If there are multiple pairs of numbers whose sum is equal to S, output the one with the smallest product of the two numbers.

## Problem-solving ideas

Use double pointers, one pointer points to the smaller value of the element and one pointer points to the larger value of the element. Pointers to smaller elements are traversed from beginning to end, and pointers to larger elements are traversed from tail to beginning.

- If the sum of two pointers pointing to elements sum == target, then these two elements are what is sought.
- If sum \> target, move the larger element to make sum smaller;
- If sum \< target, move the smaller element so that sum becomes larger.

```java
public ArrayList<Integer> FindNumbersWithSum(int[] nums, int target) {
    int i = 0, j = nums.length - 1;

            return new ArrayList<>(Arrays.asList(nums[i], nums[j]));
        if (cur < target)
            i++;
        else
            j--;
    }
    return new ArrayList<>();
}
```
