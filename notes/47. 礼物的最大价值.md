# 47. Maximum Value of Gifts

[NowCoder](https://www.nowcoder.com/questionTerminal/72a99e28381a407991f2c96d8cb238ab)

## Title description

There is a gift placed on each square of an m\*n chessboard, and each gift has a certain value (greater than 0). Start taking the gift from the upper left corner and move one space to the right or down at a time until you end in the lower right corner. Given a chessboard, find the maximum value of the gift. For example, for the following chessboard

```
1 10 3 8
12 2 9 6
5 7 4 11
3 7 16 5
```

The maximum value of the gift is 1+12+5+7+7+16+5=53.

## Problem-solving ideas

Dynamic programming should be used to solve the problem instead of depth-first search. Depth-first search is too complicated and is not the optimal solution.

```java
public int getMost(int[][] values) {
    if (values == null || values.length == 0 || values[0].length == 0)
        return 0;
    int n = values[0].length;
    int[] dp = new int[n];
    for (int[] value : values) {
        dp[0] += value[0];
        for (int i = 1; i < n; i++)
            dp[i] = Math.max(dp[i], dp[i - 1]) + value[i];
    }
    return dp[n - 1];
}
```
