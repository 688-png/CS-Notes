# 34. Paths in binary trees that sum to a certain value

## Title description

Input a binary tree and an integer, print out all paths in the binary tree whose sum of node values is the input integer. A path is defined as a path starting from the root node of the tree and going down to the nodes passing through the leaf nodes.

## Problem-solving ideas

```java
private ArrayList<ArrayList<Integer>> ret = new ArrayList<>();

public ArrayList<ArrayList<Integer>> FindPath(TreeNode root, int target) {
    backtracking(root, target, new ArrayList<>());
    return ret;
}

private void backtracking(TreeNode node, int target, ArrayList<Integer> path) {
    if (node == null)
        return;
    path.add(node.val);
    target -= node.val;
    if (target == 0 && node.left == null && node.right == null) {
        ret.add(new ArrayList<>(path));
    } else {
        backtracking(node.left, target, path);
        backtracking(node.right, target, path);
    }
    path.remove(path.size() - 1);
}
```
