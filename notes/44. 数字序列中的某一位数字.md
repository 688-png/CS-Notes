# 44. A certain digit in a sequence of numbers

## Title description

The number is serialized into a string in the format of 0123456789101112131415..., find the index position of this string.

## Problem-solving ideas

```java
public int getDigitAtIndex(int index) {
    if (index < 0)
        return -1;
    int place = 1; // 1 represents the ones digit, 2 represents the tens digit...
    while (true) {
        int amount = getAmountOfPlace(place);
        int totalAmount = amount * place;
        if (index < totalAmount)
            return getDigitAtIndex(index, place);
        index -= totalAmount;
        place++;
    }
}

/**
 * The length of the string composed of place digits
 * 10, 90, 900, ...
 */
private int getAmountOfPlace(int place) {
    if (place == 1)
        return 10;
    return (int) Math.pow(10, place - 1) * 9;
}

/**
 * The starting number of place digits
 * 0, 10, 100, ...
 */
private int getBeginNumberOfPlace(int place) {
    if (place == 1)
        return 0;
    return (int) Math.pow(10, place - 1);
}

/**
 * In a string consisting of place digits, the index number
 */
private int getDigitAtIndex(int index, int place) {
    int beginNumber = getBeginNumberOfPlace(place);
    int shiftNumber = index / place;
    String number = (beginNumber + shiftNumber) + "";
    int count = index % place;
    return number.charAt(count) - '0';
}
```
