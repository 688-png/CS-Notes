## Simple Factory

### Intent

Do not expose internal details to clients when creating an object, and provide a common interface for creating objects.

### Class Diagram

A simple factory puts the instantiation operation into a separate class, and this class becomes a simple factory class, letting the simple factory class decide which specific subclass should be used for instantiation.

This decouples the implementation of the client class from the concrete subclasses. The client class no longer needs to know which subclasses there are and which subclass should be instantiated. There are often multiple client classes. If a simple factory is not used, all client classes must know the details of all subclasses. And once the subclass changes, such as adding a subclass, all client classes must be modified.

### Implementation

```java
public interface Product {
}
```

```java
public class ConcreteProduct implements Product {
}
```

```java
public class ConcreteProduct1 implements Product {
}
```

```java
public class ConcreteProduct2 implements Product {
}
```

The following Client class contains instantiation code, which is an incorrect implementation. If this kind of instantiation code exists in the client class, you need to consider putting the code into a simple factory.

```java
public class Client {

    public static void main(String[] args) {
        int type = 1;
        Product product;
        if (type == 1) {
            product = new ConcreteProduct1();
        } else if (type == 2) {
            product = new ConcreteProduct2();
        } else {
            product = new ConcreteProduct();
        }
        // do something with the product
    }
}
```

The following SimpleFactory is a simple factory implementation that is called by all client classes that need to be instantiated.

```java
public class SimpleFactory {

    public Product createProduct(int type) {
        if (type == 1) {
            return new ConcreteProduct1();
        } else if (type == 2) {
            return new ConcreteProduct2();
        }
        return new ConcreteProduct();
    }
}
```

```java
public class Client {

    public static void main(String[] args) {
        SimpleFactory simpleFactory = new SimpleFactory();
        Product product = simpleFactory.createProduct(1);
        // do something with the product
    }
}
```
