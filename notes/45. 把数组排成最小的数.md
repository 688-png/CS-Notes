# 45. Sort the array into the smallest number

## Title description

Input an array of positive integers, concatenate all the numbers in the array to form a number, and print the smallest number among all the numbers that can be concatenated. For example, if you input the array {3, 32, 321}, the smallest number that these three numbers can be printed out is 321323.

## Problem-solving ideas

It can be seen as a sorting problem. When comparing the sizes of two strings S1 and S2, what should be compared is the size of S1+S2 and S2+S1. If S1+S2 \< S2+S1, then S1 should be ranked first, otherwise S2 should be ranked first.

```java
public String PrintMinNumber(int[] numbers) {
    if (numbers == null || numbers.length == 0)
        return "";
    int n = numbers.length;
    String[] nums = new String[n];
    for (int i = 0; i < n; i++)
        nums[i] = numbers[i] + "";
    Arrays.sort(nums, (s1, s2) -> (s1 + s2).compareTo(s2 + s1));
    String ret = "";
    for (String str : nums)
        ret += str;
    return ret;
}
```
