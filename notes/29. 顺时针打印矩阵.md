## Title description

Print the values of the matrix in a clockwise direction from outside to inside. The matrix printing results in the figure below are: 1, 2, 3, 4, 8, 12, 16, 15, 14, 13, 9, 5, 6, 7, 11, 10

## Problem-solving ideas

Print layer by layer from the outside to the inside. It can be seen that each layer of printing has the same processing steps. The only difference is that the upper, lower, left and right boundaries are different. Therefore, four variables r1, r2, c1, c2 are used to store the upper, lower, left and right boundary values ​​respectively, thereby defining the current outermost layer. The order of printing the current outermost layer: print the top line from left to right -\> print the rightmost line from top to bottom - \> print the bottom line from right to left - \> print the leftmost line from bottom to top. It should be noted that the bottom line is only printed when r1 != r2, that is, the bottom line is printed when the current number of outermost lines is greater than 1. This is because when the current outermost line only has one line, continuing to print the bottom line will result in repeated printing. The same process should be done when printing the leftmost line.

```java
public ArrayList<Integer> printMatrix(int[][] matrix) {
    ArrayList<Integer> ret = new ArrayList<>();
    int r1 = 0, r2 = matrix.length - 1, c1 = 0, c2 = matrix[0].length - 1;
    while (r1 <= r2 && c1 <= c2) {
        // up
        for (int i = c1; i <= c2; i++)
            ret.add(matrix[r1][i]);
        //right
        for (int i = r1 + 1; i <= r2; i++)
            ret.add(matrix[i][c2]);
        if (r1 != r2)
            // next
            for (int i = c2 - 1; i >= c1; i--)
                ret.add(matrix[r2][i]);
        if (c1 != c2)
            // left
            for (int i = r2 - 1; i > r1; i--)
                ret.add(matrix[i][c1]);
        r1++; r2--; c1++; c2--;
    }
    return ret;
}
```
