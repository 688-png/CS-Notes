## Title description

Print the value of each node in reverse order from end to beginning.

## Problem-solving ideas

### 1. Use recursion

To print the linked list 1-\>2-\>3(3,2,1) in reverse order, you can first print the linked list 2-\>3(3,2) in reverse order, and then print the first node 1. The linked list 2-\>3 can be regarded as a new linked list. To print the linked list in reverse order, you can continue to use the solving function, that is, call yourself in the solving function. This is a recursive function.

```java
public ArrayList<Integer> printListFromTailToHead(ListNode listNode) {
    ArrayList<Integer> ret = new ArrayList<>();
    if (listNode != null) {
        ret.addAll(printListFromTailToHead(listNode.next));
        ret.add(listNode.val);
    }
    return ret;
}
```

### 2. Use head insertion method

Head insertion method, as the name suggests, inserts the node into the head: when traversing the original linked list, insert the current node into the head of the new linked list, making it the first node.

The operation of the linked list requires maintaining the successor relationship. For example, inserting a node node2 after a certain node node1, we can achieve this by modifying the successor relationship:

```java
node3 = node1.next;
node2.next = node3;
node1.next = node2;
```

In order to insert a node into the head, we introduce an auxiliary node called the head node. This node does not store a value, just to facilitate the insertion operation. Don't confuse the head node with the first node, which is the first node in the linked list that actually stores a value.

<div align="center"> <img src="https://cs-notes-1256109796.cos.ap-guangzhou.myqcloud.com/0dae7e93-cfd1-4bd3-97e8-325b032b716f-1572687622947.gif" width="420px"> </div><br>

```java
public ArrayList<Integer> printListFromTailToHead(ListNode listNode) {
    // Head insertion method to construct reverse sequence linked list
    ListNode head = new ListNode(-1);
    while (listNode != null) {
        ListNode memo = listNode.next;
        listNode.next = head.next;
        head.next = listNode;
        listNode = memo;
    }
    //Construct ArrayList
    ArrayList<Integer> ret = new ArrayList<>();
    head = head.next;
    while (head != null) {
        ret.add(head.val);
        head = head.next;
    }
    return ret;
}
```

### 3. Use stack

The stack has the characteristics of last-in-firs
t-out. When traversing the linked list, values are put into the stack in order, and the order of the last pop-up is the reverse order.

<div align="center"> <img src="https://cs-notes-1256109796.cos.ap-guangzhou.myqcloud.com/9d1deeba-4ae1-41dc-98f4-47d85b9831bc.gif" width="340px"> </div><br>

```java
public ArrayList<Integer> printListFromTailToHead(ListNode listNode) {
    Stack<Integer> stack = new Stack<>();
    while (listNode != null) {
        stack.add(listNode.val);
        listNode = listNode.next;
    }
    ArrayList<Integer> ret = new ArrayList<>();
    while (!stack.isEmpty())
        ret.add(stack.pop());
    return ret;
}
```
