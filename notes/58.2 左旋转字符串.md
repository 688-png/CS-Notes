## Title description

Split the string S into two substrings from the Kth position, and swap the positions of the two substrings.

```html
Input:
S="abcXYZdef"
K=3

Output:
"XYZdefabc"
```

## Problem-solving ideas

First flip "abc" and "XYZdef" respectively to get "cbafedZYX", and then flip the entire string to get "XYZdefabc".

```java
public String LeftRotateString(String str, int n) {
    if (n >= str.length())
        return str;
    char[] chars = str.toCharArray();
    reverse(chars, 0, n - 1);
    reverse(chars, n, chars.length - 1);
    reverse(chars, 0, chars.length - 1);
    return new String(chars);
}

private void reverse(char[] chars, int i, int j) {
    while (i < j)
        swap(chars, i++, j--);
}

private void swap(char[] chars, int i, int j) {
    char t = chars[i];
    chars[i] = chars[j];
    chars[j] = t;
}
```
