# 17. Print the number of n digits from 1 to the maximum

## Title description

Enter a number n and print out the decimal number from 1 to the largest n digits in sequence. For example, if you enter 3, it will print out 1, 2, 3 up to the maximum 3 digits, which is 999.

## Problem-solving ideas

Since n may be very large, the number cannot be represented directly as an int, but is stored as a char array.

Use backtracking to get all the numbers.

```java
public void print1ToMaxOfNDigits(int n) {
    if (n <= 0)
        return;
    char[] number = new char[n];
    print1ToMaxOfNDigits(number, 0);
}

private void print1ToMaxOfNDigits(char[] number, int digit) {
    if (digit == number.length) {
        printNumber(number);
        return;
    }
    for (int i = 0; i < 10; i++) {
        number[digit] = (char) (i + '0');
        print1ToMaxOfNDigits(number, digit + 1);
    }
}

private void printNumber(char[] number) {
    int index = 0;
    while (index < number.length && number[index] == '0')
        index++;
    while (index < number.length)
        System.out.print(number[index++]);
    System.out.println();
}
```
