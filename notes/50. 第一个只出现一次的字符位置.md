# 50. The first character position that appears only once

## Title description

Finds the first character that occurs only once in a string and returns its position. The string contains only ASCII characters.

```
Input: abacc
Output: b
```

## Problem-solving ideas

The most intuitive solution is to use HashMap to count the number of occurrences: the character is used as the key, the number of occurrences is used as the value, and the value corresponding to the key is increased by 1 each time the string is traversed. Finally, traverse this HashMap to find the character whose occurrence count is 1.

Considering the limited range of characters to be counted, an integer array can also be used instead of HashMap. ASCII code only has 128 characters, so you can use an integer array of length 128 to store the number of occurrences of each character.

```java
public int FirstNotRepeatingChar(String str) {
    int[] cnts = new int[128];
    for (int i = 0; i < str.length(); i++)
        cnts[str.charAt(i)]++;
    for (int i = 0; i < str.length(); i++)
        if (cnts[str.charAt(i)] == 1)
            return i;
    return -1;
}
```

The space complexity of the above implementation is not yet optimal. Considering that you only need to find characters that appear only once, the number of times that needs to be counted is only 0, 1, or more. This information can be stored using two bits.

```java
public int FirstNotRepeatingChar2(String str) {
    BitSet bs1 = new BitSet(128);
    BitSet bs2 = new BitSet(128);
    for (char c : str.toCharArray()) {
        if (!bs1.get(c) && !bs2.get(c))
            bs1.set(c); // 0 0 -> 0 1
        else if (bs1.get(c) && !bs2.get(c))
            bs2.set(c); // 0 1 -> 1 1
    }
    for (int i = 0; i < str.length(); i++) {
        char c = str.charAt(i);
        if (bs1.get(c) && !bs2.get(c)) // 0 1
            return i;
    }
    return -1;
}
```
