## Title description

Given an array and the size of a sliding window, find the maximum value of all values in the sliding window.

For example, if the input array is {2, 3, 4, 2, 6, 2, 5, 1} and the sliding window size is 3, then there are a total of 6 sliding windows, and their maximum values ​​are {4, 4, 6, 6, 6, 5}.

## Problem-solving ideas

Maintain a large top heap with the size of the window, and the top heap element is the maximum value of the current window.

Suppose the size of the window is M and the length of the array is N. When the window moves to the right, the elements leaving the window need to be deleted from the heap first, and the newly arrived elements are added to the heap. The time complexity of both operations is log<sub>2</sub>M, so the time complexity of the algorithm is O(Nlog<sub>2</sub>M) and the space complexity is O(M).

```java
public ArrayList<Integer> maxInWindows(int[] num, int size) {
    ArrayList<Integer> ret = new ArrayList<>();
    if (size > num.length || size < 1)
        return ret;
    PriorityQueue<Integer> heap = new PriorityQueue<>((o1, o2) -> o2 - o1); /* Large top heap */
    for (int i = 0; i < size; i++)
        heap.add(num[i]);
    ret.add(heap.peek());
    for (int i = 0, j = i + size; j < num.length; i++, j++) { /* Maintain a large top heap of size */
        heap.remove(num[i]);
        heap.add(num[j]);
        ret.add(heap.peek());
    }
    return ret;
}
```
