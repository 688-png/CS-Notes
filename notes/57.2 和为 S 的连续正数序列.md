# 57.2 A sequence of consecutive positive numbers whose sum is S

## Title description

## Title description

Output all sequences of consecutive positive numbers that sum to S. For example, a sequence whose sum is 100 is:

```
[9, 10, 11, 12, 13, 14, 15, 16]
[18, 19, 20, 21, 22].
```

## Problem-solving ideas

```java
public ArrayList<ArrayList<Integer>> FindContinuousSequence(int sum) {
    ArrayList<ArrayList<Integer>> ret = new ArrayList<>();
    int start = 1, end = 2;
    int curSum = 3;
    while (end < sum) {
        if (curSum > sum) {
            curSum -= start;
            start++;
        } else if (curSum < sum) {
            end++;
            curSum += end;
        } else {
            ArrayList<Integer> list = new ArrayList<>();
            for (int i = start; i <= end; i++)
                list.add(i);
            ret.add(list);
            curSum -= start;
            start++;
            end++;
            curSum += end;
        }
    }
    return ret;
}
```
