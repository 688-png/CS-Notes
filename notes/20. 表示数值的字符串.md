## Problem-solving ideas

Use regular expressions for matching.

```html
[]: character set
(): Group
?: Repeat 0 ~ 1 times
+: Repeat 1 ~ n times
*: Repeat 0 ~ n times
.: any character
\\.: escaped .
\\d: Number
```

```java
public boolean isNumeric (String str) {
    if (str == null || str.length() == 0)
        return false;
    return new String(str).matches("[+-]?\\d*(\\.\\d+)?([eE][+-]?\\d+)?");
}
```
