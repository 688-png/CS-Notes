## Problem-solving ideas

Suppose the length of A is a + c, and the length of B is b + c, where c is the length of the common part of the tail, it can be seen that a + c + b = b + c + a.

When the pointer accessing the linked list A accesses the end of the linked list, let it start accessing the linked list B again from the head of the linked list B; similarly, when the pointer accessing the linked list B accesses the end of the linked list, let it start accessing the linked list A again from the head of the linked list A. In this way, the pointers accessing the two linked lists A and B can be controlled to access the intersection point at the same time.

```java
public ListNode FindFirstCommonNode(ListNode pHead1, ListNode pHead2) {
    ListNode l1 = pHead1, l2 = pHead2;
    while (l1 != l2) {
        l1 = (l1 == null) ? pHead2 : l1.next;
        l2 = (l2 == null) ? pHead1 : l2.next;
    }
    return l1;
}
```
