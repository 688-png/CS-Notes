# 4. Search in two-dimensional array

## Title description

Given a two-dimensional array, each row is sorted in ascending order from left to right, and also in ascending order from top to bottom. Given a number, determine whether the number is in the two-dimensional array.

```html
Consider the following matrix:
[
  [1, 4, 7, 11, 15],
  [2, 5, 8, 12, 19],
  [3, 6, 9, 16, 22],
  [10, 13, 14, 17, 24],
  [18, 21, 23, 26, 30]
]

Given target = 5, return true.
Given target = 20, return false.
```

## Problem-solving ideas

The required time complexity is O(M + N) and the space complexity is O(1). Where M is the number of rows and N is the number of columns.

For a number in this two-dimensional array, the number smaller than it must be on the left, and the number larger than it must be below it. Therefore, by starting the search from the upper right corner, you can quickly narrow the search interval based on the size relationship between the target and the current element, reducing the elements by one row or column at a time. The search range of the current element is all elements in the lower left corner.

```java
public boolean Find(int target, int[][] matrix) {
    if (matrix == null || matrix.length == 0 || matrix[0].length == 0)
        return false;
    int rows = matrix.length, cols = matrix[0].length;
    int r = 0, c = cols - 1; // Starting from the upper right corner
    while (r <= rows - 1 && c >= 0) {
        if (target == matrix[r][c])
            return true;
        else if (target > matrix[r][c])
            r++;
        else
            c--;
    }
    return false;
}
```
