## Title description

There are five cards, among which the big and small ghosts are leprechauns, and the card face is 0. Determine whether these five cards can form a straight.

## Problem-solving ideas

```java
public boolean isContinuous(int[] nums) {

    if (nums. length < 5)
        return false;

    Arrays.sort(nums);

    // Count the number of leprous
    int cnt = 0;
    for (int num : nums)
        if(num == 0)
            cnt++;

    // Use Leizi to complete discontinuous straights
    for (int i = cnt; i < nums.length - 1; i++) {
        if (nums[i + 1] == nums[i])
            return false;
        cnt -= nums[i + 1] - nums[i] - 1;
    }

    return cnt >= 0;
}
```
