## Title description

Implement a stack containing the min() function, which returns the smallest value currently in the stack.

## Problem-solving ideas

Using an additional minStack, the top element of the stack is the smallest value in the current stack. When pushing and popping the stack, minStack also needs to be pushed and popped, so that the top element of minStack is always the smallest value in the current stack. When performing a push operation, you need to compare the pushed element with the minimum value in the current stack, and push the element with a smaller value to minStack.

```java
private Stack<Integer> dataStack = new Stack<>();
private Stack<Integer> minStack = new Stack<>();

public void push(int node) {
    dataStack.push(node);
    minStack.push(minStack.isEmpty() ? node : Math.min(minStack.peek(), node));
}

public void pop() {
    dataStack.pop();
    minStack.pop();
}

public int top() {
    return dataStack.peek();
}

public int min() {
    return minStack.peek();
}
```
