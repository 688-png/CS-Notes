# 41.1 Median in data streams

## Question link

## Title description

How to get the median in a data stream? If an odd number of values ​​are read from the data stream, the median is the value in the middle of all the values ​​sorted. If an even number of values ​​are read from the data stream, the median is the average of the two middle numbers after all values ​​are sorted.

## Problem-solving ideas

```java
/* Large top heap, stores the left half of the elements */
private PriorityQueue<Integer> left = new PriorityQueue<>((o1, o2) -> o2 - o1);
/* Small top heap, stores the right half elements, and the right half elements are larger than the left half */
private PriorityQueue<Integer> right = new PriorityQueue<>();
/* The number of elements read by the current data stream */
private int N = 0;

public void Insert(Integer val) {
    /* Insert to ensure that the two stacks are in a balanced state */
    if (N % 2 == 0) {
        /* Insert to the right half when N is an even number.
         * Because the right half of the elements are larger than the left half, but the newly inserted elements are not necessarily larger than the left half of the elements.
         * Therefore, you need to insert the element into the left half first, and then take advantage of the fact that the left half is a large top heap, take out the top element of the heap, which is the largest element, and insert the right half at this time */
        left.add(val);
        right.add(left.poll());
    } else {
        right.add(val);
        left.add(right.poll());
    }
    N++;
}

public Double GetMedian() {
    if (N % 2 == 0)
        return (left.peek() + right.peek()) / 2.0;
    else
        return (double) right.peek();
}
```
